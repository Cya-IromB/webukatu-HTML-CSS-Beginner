jsの関数やjQueryの関数について

JavaScript側の関数など

ーイベント発火用関数ー

  • 'click'
    => クリック
  • 'mousemove'
    => マウスがターゲット内に侵入した瞬間
  • 'mouseover'
    => マウスが乗ってる
  • 'mouseout'
    => マウスが外に出た瞬間
  • 'mousedown'
    => ボタンが押された
  • 'mouseup'
    => ボタンが離された

  • 'keypress'
    => キーを押して離した
  • 'keydown'
    => キーを押している
  • 'keyup'
    => キーを離した瞬間

  • 'DOMFocusIn'
    => ターゲットがフォーカスを受け取った
  • 'DOMFocusOut'
    => ターゲットがフォーカスを失った
  • 'DOMActive'
    => ターゲットがアクティブになった

  • 'select'
    => テキストフィールドで文字が選択された
  • 'change'
    => コントロールの値が変化した
  • 'submit'
    => submitが押された
  • 'reset'
    => resetボタンが押された
  • 'focus'
    => コントロールがフォーカスを受け取った
  • 'blur'
    => コントロールがフォーカスを失った。

  • 'load'
    => 読み込みが完了した時
  • 'scroll'
    => コントロール付属のスクロールバー位置が変更された時
  • 'resize'
    => コントロールのサイズが変更されたときに発火

  • 'touchstart'
    => タッチ開始されたとき
  • 'touchmove'
    => 指を画面上で動かしているとき
  • 'touchend'
    => 指を画面から離したとき

addEventListenerの第二引数のfunction(e)に使えるもの

  • e.target
    => 対象の要素
  • e.clientX
    => ブラウザ画面内のX座標
  • e.clientY
    => ブラウザ画面内のY座標
  • e.pageX
    => HTML全体のX座標
  • e.pageY
    => HTML全体のY座標
  • e.offsetX
    => ターゲット内のX座標
  • e.offsetY
    => ターゲット内のY座標

jQuery側の関数など

-基礎構文-

        jQuery(function(){
          $('対象セレクター').条件(function({
            $(this).メソッド(引数).
            $(this).メソッド(引数).
            $(this).メソッド(引数);
          });
        });

ーイベントの内容ー

  • addClass('任意のクラス名');
    => クラスを追加する
  • attr('取得する属性','属性の内容');
    => 属性の中身を書き換える
  • hide();
    => 要素を非表示にする
  • show();
    => 要素を表示する
  • toggle();
    => 非表示、表示を切り替える
  • text('書き換える内容')
    => 要素内のテキストを引数の内容に書き換える。
